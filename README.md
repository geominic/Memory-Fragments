Deskripsi Game
Memory Fragments adalah game 2D side-scrolling platformer dengan gaya visual neon retro yang menggabungkan aksi parkour cepat dengan tantangan memory puzzle.
Pemain akan menjelajahi berbagai level yang dipenuhi rintangan, jebakan, dan musuh. Untuk bertahan hidup, pemain harus menggunakan kemampuan gerakan seperti wall jump, wall slide, dan berbagai manuver parkour untuk menghindari bahaya.
Di sepanjang level, pemain dapat menemukan Memory Game yang harus diselesaikan untuk memperoleh Memory Fragments. Fragmen ini merupakan item penting yang dibutuhkan untuk membuka pintu keluar level dan melanjutkan permainan.
Permainan menuntut refleks, ketepatan gerakan, dan kemampuan mengingat, sehingga menciptakan pengalaman bermain yang menantang sekaligus strategis.

Memory Game & Memory Fragments
Memory Fragments merupakan item penting yang harus dikumpulkan untuk menyelesaikan level.
Cara Mendapatkan Fragment
Fragment diperoleh dari Memory Match Minigame yang tersebar di level.
Memory Match
Puzzle mencocokkan kartu berukuran 4 × 4.
Pemain harus menemukan pasangan kartu yang sama.
Task Object
Beberapa level memiliki Task (objek interaktif) yang:
Membuka minigame Memory Match
Memberikan Memory Fragment setelah diselesaikan
Memutar completion sound sebagai tanda keberhasilan
Persyaratan Level
Setiap level memiliki Exit Door dengan syarat jumlah fragment tertentu.
Pintu tidak dapat dibuka jika jumlah fragment belum mencukupi.
Pemain harus menyelesaikan Memory Game yang tersedia untuk mengumpulkan fragment yang diperlukan.

Extra Rewards
Selain memenuhi persyaratan fragment untuk membuka pintu keluar, pemain juga dapat memperoleh reward tambahan.
Jika pemain mengumpulkan Memory Fragment melebihi jumlah minimum yang dibutuhkan untuk menyelesaikan level, pemain akan mendapatkan 1 poin One-Time Shield sebagai hadiah.
One-Time Shield
Melindungi pemain dari 1 kali kematian akibat musuh atau jebakan.
Shield akan aktif secara otomatis saat pemain terkena fatal damage.
Bersifat sekali pakai.
Shield yang diperoleh dapat ditabung dan digunakan pada level berikutnya.
Sistem ini memberikan insentif bagi pemain untuk mengeksplorasi level lebih jauh dan menyelesaikan lebih banyak Memory Game daripada yang diwajibkan.
